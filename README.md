# MarketPro - Sistema de Gestão de Mercado

## Nova funcionalidade adicionada
Foi adicionada a funcionalidade de **filtro e exportação CSV do histórico de vendas**.

### O que ela faz
- filtra vendas por cliente
- filtra vendas por data
- exporta o histórico filtrado em CSV

## Estrutura
- `frontend/index.html`
- `backend/app.js`
- `backend/package.json`
- `.gitignore`

## Como rodar
### Backend
```bash
cd backend
npm install
npm start
```

### Frontend
Abra `frontend/index.html` no navegador.
